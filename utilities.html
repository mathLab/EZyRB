Module implementing some utilities for EZyRB

compute_area(filename)[source]

Given a file, this method computes the area for each cell of the mesh stored in the file and returns it. It uses polygon_area().

Parameters:filename (str) – the name of the file to parse in order to extract the necessary information about the cells.
Returns:the array that contains the area of each cells.
Return type:numpy.ndarray
compute_normals(filename, datatype='cell')[source]

Given a file, this method computes the surface normals of the mesh stored in the file. It allows to compute the normals of the cells or of the points. The normal computed in a point is the interpolation of the cell normals of the cells adiacent to the point.

Parameters:
  • filename (str) – the name of the file to parse in order to extract the geometry information.
  • datatype (str) – indicate if the normals have to be computed for the points or the cells. The allowed values are: ‘cell’, ‘point’. Default value is ‘cell’.
Returns:

the array that contains the normals.
Return type:

numpy.ndarray

Parameters:simplex_vertices (numpy.ndarray) – Nx3 array containing the parameter values representing the vertices of a simplex. N is the dimensionality of the parameters.

write_area(filename, output_name='Area')[source]

Given a file, this method computes the area for each cell of the mesh stored in the file and save it as new dataset.

Parameters:
  • filename (str) – the name of the file to parse in order to extract the geometry information.
  • output_name (str) – the name of the new dataset that contains the cells area.
write_normals(filename, output_name='Normals', datatype='cell')[source]

Given a file, this method computes the surface normals of the mesh stored in the file and save it as new dataset.

Parameters:
  • filename (str) – the name of the file to parse in order to extract the geometry information.
  • output_name (str) – the name of the new dataset that contains the normals.
  • datatype (str) – indicate if the normals have to be computed for the points or the cells. The allowed values are: ‘cell’, ‘point’. Default value is ‘cell’.
